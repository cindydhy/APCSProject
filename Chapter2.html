Chapter 2: Primitive Data and Definite Loops


Data types

  • Data type: name for a category of data values that are all related, as in type int (used to represent integer values)
    • Primitive types: boolean, byte, char, double, float, int, long, short
      • What is the difference between types float and double?
        • float can handle about 7 decimal places whereas double can handle about 16 decimal places. Type double gives you more precision.
        • float is represented in 32 bits, with 1 sign bit while double is represented in 64 bits, with 1 sign bit.
      • Objects: string, point

Rules of precedence

  • Precedence: binding power of an operator, which determines how to group parts of an expression
    • Java operator precedence:
      • Unary operators:   +, -
      • Multiplicative operators:   *, /, %
      • Additive operators:   +, -
    • The use of parentheses can force a certain evaluation
      • Ex.: (4  +  5)  *  -5  =  -45 instead of -21

Mixing types and casting

Increment/decrement operators

  • Special assignment characters:
    • x +=  1;   →   x   =  x + 1;
    • x -=  1;   →   x   =  x - 1;
    • x *=  1;   →   x   =  x * 1;
    • x /=  1;   →   x   =  x / 1;
    • x %=  1;   →   x   =  x % 1;

  • What is the difference between x++ and ++x?
    • ++x is known as preincrement. This means that 1 is added to x before any operations or comparisons is carried out.
    • x++ is known as postincrement. This means that 1 is added to x after any operations or comparisons is carried out.
    • Ex.: int x = 8;
      System.out.println(x++);    // prints 8 but x   =   9
      int a = 8;
      System.out.println(++a);    // prints 9 and a   =   9
