Chapter 7: Arrays


Array basics

  • Array: object that stores many values of the same type
  • Index: 0-based integer to access an element from an array
  • Zero-based indexing: numbering scheme used throughout Java in which a sequence of values is indexed starting with 0
  • Element: one value in an array
  • Constructing an array
    • General syntax: type[] name = new type[lenght];
      • Ex.: int[] numbers = new int[10];
    • Auto-initialization: initialization of variables to a default value, such as on an array's elements when it is constructed
    • Each element initially gets a "zero-equivalent" value
    • Array-traversal: processing each array element sequentially from the first to the last
    • Quick array initialization: type[] name = {value, value, value,...value};



  • Accessing elements
    • General syntax: name[index]
      • Ex.: numbers[0] = -8;
      • int x = numbers[9];
      • numbers[numbers[4]] = x;
    • Out-of-bounds
      • Legal indexes: between 0 and array's length - 1
        • Negatives and any number greater than length - 1 are illegal and will throw an exception
  • Sequential access: manipulating values in a sequential manner from first to last
  • Arrays allow random access
    • Random access: manipulating values in order whatsoever to allow quick access to each value
  • Limitations of arrays
    • Cannot resize an existing array
    • Cannot compare arrays with == or equals
    • Does not know how to print itself
      • Ex.: int[] list = {42, 42, 50, 18, 4};
               System.out.println(list);
               This prints [I@98f8c4], which is the array's reference
    • To print an array use toString method
  • When you pass an array as a parameter to a void method, the method has the ability to change the array's contents.



The for-each loop

  • General syntax
    • for (type name : array){
           statement;
      statement;
      }

Reference semantics

  • Value semantics (value types): behavior where values are copied when assigned, passed as parameters, or returned
  • Reference semantics (reference types): behavior where variables actually store the address of an object in memory
  • Arrays are passed as parameters by reference
  • Null: a Java keyword signifying no object

Multidimensional arrays

  • Multidimensional array: an array of arrays, the elements of which are accessed with multiple integer indexes
  • General syntax: type[][]
    • double[row][column]
    • Ex.: double[][] lists = new double[4][9];
