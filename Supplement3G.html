Supplement 3G Graphics


Drawing panel

  • DrawingPanel class link
  • Graphics belongs to a package named Abstract Window Toolkit (AWT)
  • Pixel: single small dot on a computer screen
  • Graphical objects
    • DrawingPanel: window on the screen
    • Graphics: a "pen" to draw shapes and lines on a window
    • Color: colors in which to draw shapes
    • To create a window:
      • DrawingPanel name = new DrawingPanel(width, height);
      • Ex.: DrawingPanel panel = new DrawingPanel(300, 200);
    • To access graphics:
      • Graphics g = panel.getGraphics();
    • Drawing shapes:
      • g.fillRect(10, 30, 60, 35);
        g.fillOval(80, 40, 50, 70);
  • Java class libraries: classes included with Java's JDK
    • Import declaration: import pacakageName.*;
    • Import declarations must be placed above the line before the public class header
    • Graphics belongs to a package named java.awt
