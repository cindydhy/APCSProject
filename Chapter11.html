Chapter 11: Java Collections Framework


Definitions

  • Collection: an object that stores a group of other objects, called its elements
  • List: ordered collection of elements, often accessed by integer indexes or by iteration
  • Stack: collection in which the last element added is the first one to be removed
  • Queue: collection in which elements are removed in the same order in which they were added
  • Set: collection of elements that is guaranteed to contain no duplicates
  • Map: collection of key/value pairs in which each key is associated with a corresponding value
