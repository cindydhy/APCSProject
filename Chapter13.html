Binary search

  • Binary search: an algorithm that searches for a value in a sorted list by repeatedly dividing the search space in half

Program complexity

  • Complexity: a measure of the computing resources that are used by a piece of code, such as time, memory, or disk space
  • Complexity class: a set of algortihms that have a similar relationship between input data size and resource consumption
